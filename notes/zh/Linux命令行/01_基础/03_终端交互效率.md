# ⌨️ 编辑器与终端交互 (Efficiency)

### 1. Vim 编辑器
Vim 是一种强大的文本编辑器，主要有三种模式：
- **普通模式** (默认)：进行移动、删除、复制等操作
  - `i`：进入插入模式
  - `x`：删除当前字符
  - `dd`：删除当前行
  - `yy`：复制当前行
  - `p`：粘贴
  - `u`：撤销
  - `/word`：向后搜索 "word"，按 `n` 查找下一个，`N` 查找上一个
  - `Esc`：回到普通模式
- **插入模式**：进行文本编辑
- **命令模式** (普通模式下按 `:` )：
  - `:w`：保存
  - `:q`：退出
  - `:wq`：保存并退出
  - `:q!`：强制退出且不保存

### 2. Nano 编辑器 (极简)
Nano 比 Vim 更简单，操作提示直接显示在底部，适合临时修改小文件：
- `nano file`：打开或创建文件
- `Ctrl + O`：保存文件
- `Ctrl + X`：退出 (若未保存会询问)
- `Ctrl + W`：搜索文本

### 3. 终端交互快捷键 (Terminal Shortcuts)
掌握这些快捷键，你的终端操作效率将提升一个量级。

#### 文本编辑快捷键
- **`Ctrl + A` / `Ctrl + E`**：光标移动到行首/行尾。
- **`Ctrl + U` / `Ctrl + K`**：剪切/删除光标前/后的所有内容。
- **`Ctrl + W`**：删除光标前的一个单词。
- **`Ctrl + L`**：**清屏** (等同于执行 `clear` 命令)。
- **`Ctrl + C`**：终止当前进程。
- **`Ctrl + D`**：退出当前 Shell 或退出当前登录。

#### 历史记录操作 (History Expansion)
- `!!`：执行上一条命令 (如 `sudo !!`)。
- `!$`：引用上一条命令的**最后一个参数**。
- **`!^`**：引用上一条命令的**第一个参数**。
- **`!n`**：执行历史记录中第 **n** 条命令。
- **💡 效率组合：快捷键版**
  - **`Alt + .`**：快速粘贴上一条命令的最后一个参数 (比 `!$` 更快，可多次按切换)。
  - **`Ctrl + R`**：**搜索历史命令** (Reverse-i-search)。输入关键词即可匹配。推荐安装 `fzf` 替代默认搜索，体验飞升。
  - **`Ctrl + X, Ctrl + E`**：**在编辑器中编辑长命令**。当你写了五行长的命令发现第一行错了，按它，会在 Vim 里打开这行命令，改完保存自动执行。

### 4. 别名管理 (Alias)
通过 `alias` 将长命令变短，配置在 `~/.bashrc` 或 `~/.zshrc` 中可永久生效。

- `alias ll='ls -lha'`：列出所有文件的详细信息。
- `alias grep='grep --color=auto'`：让 grep 输出带颜色。
- `alias update='sudo apt update && sudo apt upgrade'`：一键更新系统。
- `alias ..='cd ..'`：快速返回上级目录。
